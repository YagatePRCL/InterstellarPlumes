# InterstellarPlumes
A collection of Waterfall Plumes suited for Interstellar Technologies in KSP

**InterstellarPlumes is still in BETA. Plumes may change between updates**

![](https://i.imgur.com/I6qYQD0.png)
(an example InterstellarPlumes on Tweakscaled Nertea BCAM engines. Craft made by YGTE)

---

## InterstellarPlumes is a collection of Waterfall configs that improve upon the pre-existing ones that come included with most KSP Far Future mods.
### Current list of planned support:
- Far Future Technologies
- Sterling Systems
- Other contending mods, upon request.
### Current list of features/goals
- Mimic the style of KSP 2 plumes
- Plumes that appear visually striking/TUFX.
- Accurate plume options for Interstellar feasible engines.

---
### Recommended mods/Plume Support:
- Far Future Technologies
- Sterling Systems

### INSTALLATION:

Drag and drop the folder inside the extracted () called () inside Gamedata, into your actual KSP Gamedata
